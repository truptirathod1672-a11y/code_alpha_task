 Project Overview

This project is part of the CodeAlpha Data Science Internship.
It focuses on analyzing unemployment trends in India using Python.

The goal is to understand unemployment patterns, study the impact of Covid-19, and extract insights that may help in economic and policy decision-making.

 Project Objective

Analyze unemployment rate data

Perform data cleaning and preprocessing

Visualize unemployment trends

Study Covid-19 impact

Identify seasonal and regional patterns

Generate actionable insights

* Dataset

Unemployment in India Dataset (CSV format)

Features include:

State

Date

Frequency

Estimated Unemployment Rate

Estimated Employed

Labour Participation Rate

Region

* Tools & Libraries

Python

Jupyter Notebook

Pandas

NumPy

Matplotlib

Seaborn

*Project Workflow

Load dataset

Rename and clean columns

Handle missing values

Convert Date column to datetime format

Perform Exploratory Data Analysis (EDA)

Visualize overall unemployment trends

Analyze Covid-19 period impact

Identify monthly and state-wise patterns

* Key Insights

Significant spike in unemployment during Covid-19 (2020).

Urban regions showed higher fluctuation compared to rural regions.

Some states consistently recorded higher unemployment rates.

Monthly seasonal trends observed across the dataset.
